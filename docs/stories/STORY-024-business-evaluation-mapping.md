## Epic

EPIC-003

## Functional Requirements

- FR-028
- FR-022

## User Story

As a harness trainer, I want business success criteria to map into evaluator
checks so that harness training improves outcomes that matter to the target
business Agent.

## Implementation Plan

1. Define mapping from business success criteria to evaluator checks.
2. Define regression fixtures for key workflows.
3. Define failure classes such as incorrect tool use, missing approval, policy
   violation, unsupported data access, budget overrun, and poor task outcome.
4. Add result fields for keep, discard, and crash decisions.

## Acceptance Criteria

- Business success criteria have evaluator coverage.
- Acceptance criteria map to deterministic checks where possible.
- Evaluator output can support keep/discard/crash decisions.
